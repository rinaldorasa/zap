import base64
import io
from PIL import Image

# Convertire l'immagine PNG in base64
image = Image.open("image.png")
byte_array = io.BytesIO()
image.save(byte_array, format="PNG")
base64_string = base64.b64encode(byte_array.getvalue())

# Convertire la stringa base64 in un'immagine WebP
webp_image = Image.open(io.BytesIO(base64.b64decode(base64_string)))
webp_image.save("image.webp")

// Convertire l'immagine PNG in base64
function encodeImageToBase64(image) {
  var canvas = document.createElement("canvas");
  var ctx = canvas.getContext("2d");
  ctx.drawImage(image, 0, 0);
  var dataURL = canvas.toDataURL("image/png");
  return dataURL;
}

// Convertire la stringa base64 in un'immagine WebP
function decodeImageFromBase64(dataURL) {
  var img = new Image();
  img.src = dataURL;
  return img;
}

// Utilizzare le funzioni precedenti per convertire un'immagine PNG in un'immagine WebP

document.getElementById("webpImage").appendChild(webpImage);

La funzione window.scrollTo() accetta due argomenti: la posizione X e la posizione Y in cui si desidera scorrere la finestra. Per scorrere fino alla fine della pagina, puoi utilizzare il seguente codice:

window.scrollTo(0, document.documentElement.scrollHeight);

window.location.reload();

window.location.href = "https://www.google.com";

file_put_contents('testo.txt', $Testo);

$file = fopen('testo.txt', 'w');

Sì, esistono diversi modi per aggiornare una pagina in PHP o JavaScript.

In PHP, puoi usare la funzione header() per inviare un header HTTP al browser, che gli chiederà di aggiornare la pagina corrente. Il seguente codice aggiornerà la pagina corrente:

header("Refresh: 0");
L'argomento 0 della funzione header() indica che la pagina deve essere aggiornata immediatamente. Puoi anche usare un valore diverso per l'argomento 0 per ritardare l'aggiornamento della pagina. Ad esempio, il seguente codice aggiornerà la pagina corrente dopo 5 secondi:

header("Refresh: 5");
In JavaScript, puoi usare la funzione window.location.reload() per aggiornare la pagina corrente. Il seguente codice aggiornerà la pagina corrente:

window.location.reload();
Puoi anche usare la funzione window.location.href per aggiornare la pagina corrente a una nuova URL. Il seguente codice aggiornerà la pagina corrente alla URL https://www.google.com:

window.location.href = "https://www.google.com";
